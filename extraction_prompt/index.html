리니지M 계정 데이터 추출
범용 AI 프롬프트

바로템 거래완료 페이지 스크린샷 + 텍스트에서 시세 DB 및 상품 리포트용 데이터를 추출합니다.
Claude, GPT-4o, Gemini 등 모든 AI에서 동일하게 동작합니다.

Claude · GPT-4o · Gemini 호환 전체 스크린샷 1장으로 동작 리니지M 게임 내 등급 기준 DB + 리포트 동시 추출

사용 방법

아래 프롬프트를 복사해서 AI에 붙여넣은 뒤, 스크린샷과 텍스트를 함께 제공합니다.

STEP 01
파일 준비
상품 3개분 파일 준비
텍스트 3개 + 이미지 3개 = 총 6개
STEP 02
프롬프트 복사
아래 전체 프롬프트를 복사합니다.
STEP 03
AI + 파일 첨부
Claude / GPT-4o / Gemini에 프롬프트 붙여넣기 후 6개 파일 한 번에 첨부
STEP 04
결과 확인
분석 결과 확인. 이상 없으면 다음 단계로.
STEP 05
즉시 저장
저장용 테이블을 엑셀에 복사·붙여넣기. 분석할 때마다 바로 저장.

바로템 페이지 섹션 구조

전체 스크린샷은 아래 순서로 섹션이 세로 나열됩니다. 섹션 순서를 기준으로 읽으세요.

순서섹션명추출 항목위치 특징
1상품 기본 정보클래스·서버·거래완료가·레벨최상단. 텍스트 정보 중심. 레벨은 캐릭터 초상화 왼쪽 하단 원형 뱃지 안 숫자.
2장비 정보이동악세 수량·각인 여부·문양 완성도캐릭터 전신 + 장비 슬롯 패널. 가로로 아이콘 나열.
3~N변신 패널 (반복)변신 등급·각성 여부·옵션원형 아이콘 1개가 중앙에 크게. 주변에 작은 원들이 방사형 배치.
N+1인형 패널 (반복)인형 등급·각성 여부·옵션변신 패널과 동일 구조. 아이콘 형태 다름.
N+2성물 패널 (반복)성물 등급·각성 여부변신 패널과 동일 구조.
N+3스킬 목록신화스킬 수·전설스킬 수표 형태. 스킬명과 등급 아이콘 나열.
N+4컬렉션·문양컬렉션 진행도·문양 완성도최하단부.

📋 전체 프롬프트 (복사해서 사용)

아래 전체 내용을 복사해서 AI에 붙여넣은 뒤 스크린샷과 텍스트 파일을 첨부합니다.

[역할] 너는 리니지M 계정 거래 데이터 분석 전문가다. 바로템 거래완료 페이지의 스크린샷(이미지)과 텍스트를 함께 분석해서 아이템매니아 상품 추천 시스템의 시세 DB와 리포트 생성을 위한 구조화된 데이터를 추출한다. [목적] 1. 시세 기준값 구축: 거래완료 데이터를 누적해 클래스별 최저·평균·최고가 산출 2. 리포트 생성: 상품 상세 리포트(가격 위치·스펙 평가·활용 가능성)에 필요한 문장 생성 [입력 데이터] 한 번에 상품 3개분 파일을 제공한다. (최대 5개까지 가능) - 텍스트 3개: 거래번호·거래완료가·서버·클래스·조회수·인증상태·판매자 설명 포함 - 이미지 3개: 바로템 거래완료 페이지 전체 스크롤 캡처 파일명 규칙: 파일명이 같고 확장자만 다른 것이 같은 상품이다. 예) 260309_brt_0001.txt + 260309_brt_0001.png → 상품 1번 260309_brt_0002.txt + 260309_brt_0002.png → 상품 2번 260309_brt_0003.txt + 260309_brt_0003.png → 상품 3번 [이미지 분석 원칙 — 반드시 준수] 이미지는 바로템 거래완료 페이지를 전체 스크롤 캡처한 것이다. 전체 이미지를 한 번에 읽으려 하지 말고, 아래 섹션 순서대로 섹션 하나씩 집중해서 세밀하게 읽어야 한다. 이미지 해상도가 낮을 수 있으므로 각 섹션의 아이콘 위치·색상·개수를 최대한 정밀하게 파악한다. 확인이 불가한 항목은 반드시 "확인필요"으로 표기하고 절대 임의 추정하지 않는다. [레벨 확인 방법 — 중요] 레벨은 아래 위치에 표시된다. 반드시 이 부분을 집중해서 읽어라. 위치: 이미지 상단부 섹션 1 내, 캐릭터 초상화(얼굴 이미지) 왼쪽 하단 형태: 원형 뱃지(동그라미) 안에 숫자로 표시 예시: 원 안에 "89"라고 적혀 있으면 레벨 89 주의: 캐릭터 초상화 위쪽이나 오른쪽이 아닌 반드시 왼쪽 하단 원형 안 숫자 [바로템 페이지 섹션 구조 — 섹션별 순서대로 읽기] 이미지는 세로로 스크롤되는 전체 페이지 캡처다. 섹션이 가로선으로 구분되어 아래 순서로 나열된다. 반드시 섹션 1부터 순서대로 하나씩 집중해서 읽는다. 섹션 1 — 상품 기본 정보 (최상단) 추출 항목: 클래스 / 서버 / 거래완료가 레벨: 캐릭터 초상화 왼쪽 하단 원형 뱃지 안 숫자 (위 [레벨 확인 방법] 참고) 읽기 방법: 텍스트가 중심이므로 텍스트 파일과 함께 교차 확인 섹션 2 — 장비 정보 추출 항목: 이동악세 수량 / 각인 여부 / 문양 완성도 특징: 캐릭터 전신 + 장비 슬롯 아이콘이 가로로 나열 읽기 방법: 악세서리 슬롯 위치의 아이콘 수량을 세어 이동악세 수량 파악 섹션 3~N — 변신 패널 (섹션마다 1개씩 반복) 추출 항목: 변신 등급 / 각성 완료 여부 / 각성 옵션 특징: 원형 아이콘 1개가 중앙에 크게, 주변 작은 원 방사형 배치 읽기 방법: 아래 [색상 → 등급 변환 규칙] 적용하여 등급 판단 섹션 N+1 — 인형 패널 (반복) 추출 항목: 인형 등급 / 각성 완료 여부 / 옵션 읽기 방법: 변신 패널과 동일하게 색상으로 등급 판단 섹션 N+2 — 성물 패널 (반복) 추출 항목: 성물 등급 / 각성 완료 여부 읽기 방법: 변신 패널과 동일하게 색상으로 등급 판단 섹션 N+3 — 스킬 목록 추출 항목: 신화스킬 수 / 전설스킬 수 특징: 표 형태로 스킬명·등급 아이콘 나열 읽기 방법: 아이콘 색상으로 신화스킬·전설스킬 구분 후 각각 개수 집계 섹션 N+4 — 컬렉션·문양 (최하단) 추출 항목: 컬렉션 진행도 / 문양 완성도 [색상 → 등급
